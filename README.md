# Zepto Inventory Data Analysis Using SQL

## Project Overview

This is a beginner-friendly Data Analytics project where I analyzed a Zepto inventory dataset using SQL.  
The main goal of this project is to extract useful business insights related to product pricing, discounts, stock availability, inventory management, and category-wise performance.

The project demonstrates how SQL can be used to analyze real-world e-commerce inventory data and answer important business questions.

---

## Dataset

The dataset contains product inventory details from Zepto including:

- Product Name
- Category
- MRP
- Discount Percentage
- Available Quantity
- Discounted Selling Price
- Product Weight
- Stock Availability

Dataset Source: Kaggle

## Business Questions Solved

1. Find the top 10 best-value products based on discount percentage.

2. Find high MRP products that are currently out of stock.

3. Calculate estimated revenue for each category.

4. Find products where MRP is greater than ₹500 and discount is less than 10%.

5. Identify categories offering highest average discount percentage.

6. Calculate price per gram for products to find best value products.

7. Group products into Low, Medium, and Bulk categories based on weight.

8. Find total inventory weight per category.

---

## Key Insights

- Identified products with highest discounts.
- Found expensive products unavailable in inventory.
- Analyzed revenue contribution by category.
- Compared product value using price per gram.
- Created inventory weight categories.
